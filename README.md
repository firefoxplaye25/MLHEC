# Mlh Compress Extract

An advanced compression and extraction utility for the custom `.mlhpkg` format, designed with high performance and a classic Windows aesthetic in mind.

## Features

- **High-Performance Compression**: Specialized engine for `.mlhpkg` files.
- **Single-File Portable**: No external dependencies required for execution.
- **Windows Shell Integration**: Easily compress and extract files via the context menu.
- **Classic UI**: A clean, retro-modern interface inspired by the Windows 7 era.
- **Integrated Installer**: Built-in logic for easy deployment and integration.

## Getting Started

### Prerequisites

- .NET 10.0 SDK

### Building

1. Clone the repository.
2. Open the project folder.
3. Run `dotnet build` or open the project in Visual Studio.
